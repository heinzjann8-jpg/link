Pizza Ontology Project

by Heinrick Janiola
Pizza ontology that took about 2 full days to configure and understand unfamiliar concepts; implements and features SWRL, I also adopted ordered enumerated classes, inferencing, inversed object properties, cardinal, existential and universal restriction with some help and understanding from tutorials. by Heinrick Janiola. Reasoner used was HermiT 1.4.3.456

Class Hierarchy

Here are the main classes in my pizza ontology:

  • Pizza - Main class for all pizzas
    • NamedPizza - Specific pizza types
      • MargheritaPizza
      • olivePizza
      • pepperoniPizza
      • spicyPepperoniPizza
    • MeatPizza
    • VegetarianPizza (defined by reasoner)
    • SpicyPizza (defined by reasoner)
    • InterestingPizza (defined by reasoner)
    • pizzaThatHasCheese (defined by reasoner)
    • pizzaLotsOfCalorie (defined by reasoner)
    • pizzaLowCalorie (defined by reasoner)
  • PizzaBase - Types of pizza crust
    • ThinAndCrispyBase
    • DeepPanBase
  • PizzaTopping - Pizza ingredients
    • CheeseTopping
      • MozzarellaTopping
      • ParmesanTopping
    • MeatTopping
      • HamTopping
      • PepperoniTopping
      • SalamiTopping
      • SpicyBeefTopping
    • SeafoodTopping
      • AnchovyTopping
      • PrawnTopping
      • TunaTopping
    • VegetableTopping
      • CaperTopping
      • MushroomTopping
      • OliveTopping
      • TomatoTopping
      • PepperTopping
        • GreenPepperTopping
        • JalapenoPepperTopping
        • RedPepperTopping
  • Spiciness - Enumerated class with three values: Hot, Medium, Mild

Properties

Object Properties:

hasBase
Domain: Pizza
Range: PizzaBase
Description: Connects a pizza to its base type
hasTopping
Domain: Pizza
Range: PizzaTopping
Description: Connects a pizza to its toppings
hasSpiciness
Domain: PizzaTopping
Range: Spiciness
Description: Shows how spicy a topping is

Data Properties:

hasCaloricContent
Domain: Pizza
Range: xsd:integer
Description: The number of calories in a pizza

Instances

Spiciness Levels: Hot, Medium, Mild

Example Pizzas: MargheritaPizza (mozzarella + tomato), olivePizza (olives + mozzarella + parmesan + tomato), pepperoniPizza (pepperoni + mozzarella + tomato), spicyPepperoniPizza (adds jalape√±o peppers)

Key Restrictions

  • Every Pizza must have at least one PizzaBase
  • Every Pizza must have a caloric content value
  • VegetarianPizza can only have cheese or vegetable toppings
  • SpicyPizza has at least one topping with "Hot" spiciness
  • InterestingPizza has 3 or more toppings
  • pizzaLotsOfCalorie has 400+ calories
  • pizzaLowCalorie has less than 400 calories
  • MargheritaPizza can only have mozzarella and tomato (nothing else)
  • JalapenoPepperTopping always has "Hot" spiciness
  • Disjoint: Toppings cannot be both cheese and meat, or meat and seafood, etc.
